**DEV.PORTO** is a specialized engineering portfolio designed to showcase a unique intersection of creative design and robust software engineering. Built as a high-fidelity Single Page Application (SPA), it utilizes modern UI patterns like glassmorphism and glow-effects to present a professional digital persona.

---

## 🎨 Visual Language
- **Glow-Centric UI:** Utilizes CSS `box-shadow` and `backdrop-filter` to create a deep, layered technological aesthetic.
- **Glassmorphism:** Implementing the "Glass-Panel" approach for project cards to ensure a clean, breathable interface.
- **Micro-Animations:** Background floating orbs and hover-scaling elements provide a sense of life and interactivity.

---

## 🛠️ Key Capabilities
- **🎭 Portfolio Grid:** A dynamic, data-driven showcase of engineering projects.
- **📱 Fluid Responsiveness:** Engineered for a consistent, premium experience across mobile, tablet, and desktop viewports.
- **⚡ Performance Optimized:** Low-latency load times powered by Vite's ESM-based build system.

---

## 🏗️ Technical Stack
- **React 18:** Modern functional architecture for modular UI development.
- **Advanced CSS:** Custom animations, keyframes, and specialized variables for the "Glow" theme.
- **Vite:** Next-generation frontend tooling.

---

git clone https://github.com/DH-YASSER/developer-portfolio.git
